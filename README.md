🚌 Bus Ticket Reservation System (Java Console Application)
This is a professional Java-based console application designed to manage bus ticket reservations.
The project demonstrates core Object-Oriented Programming (OOP) principles, data structures, and complex business logic for a real-world scenario.

🚀 Features
Dynamic Bus Layouts: Generates seat maps dynamically based on seat formats (2+1 or 2+2) and custom numbering styles.

Intelligent Pricing: Calculates ticket prices automatically based on passenger age (Child, Student, Senior discounts) and trip type (One-way or Round-trip).

Real-time Availability: Users can view live seat maps where [o] represents available seats and [x] represents reserved ones.

Trip Filtering: Allows users to search and filter trips by origin, destination, and date.

🛠️ Technical Stack
Language: Java 17
Concepts: Encapsulation, Enums, List & Map Collections.

(Personal Note)
I built this project as a hands-on way to master Object-Oriented Programming (OOP).
Instead of just reading theory, I wanted to see how real-world objects like Bus, Trip, and Ticket work together in a complete system.

In this project, I focused on writing clean, maintainable, and well-structured code.

ensuring a robust user experience even in a console environment.
